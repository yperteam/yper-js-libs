# yper-js-libs
Yper JS Libs

## To install dependencies
```bash
npm install
```

## To build
```bash
npm run build
```

## To add to your project
```bash
yarn add yperteam/yper-js-libs
```

## To load a specific library in your project (example: LeafletHelper)
```js
import {LeafletHelper} from "yper-js-libs/dist/yper_libs";
```
